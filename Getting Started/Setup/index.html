Getting Started ➜ Bot Setup

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Once the bot is successfully invited to your server, you can proceed with setting up and configuring the bot. For initial server setup use /server-setup. After that, verify the the setup has been successfully completed by running /rank, if the server setup was successful the bot will answer with your rank card. If you successfully setup your server you can proceed to the next step!
