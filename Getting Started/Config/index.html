Getting Started ➜ Configuration

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Polaris operates its events based on channel descriptions. For correct usage of all the features, configure your channel topics to the ones below. Multiple features cannot be used in the same channel. For log channel configuration use the second table.

Event Channel Topic Log Event Channel Topic
Verification pverify Banning pban
Leveling Messages plevels Kicking pkick
Suggestions psuggest Warns pwarn
Reports preport Invites pinvite
Reviews preview Reports preports
Music Commands pmusic Muting pmute
Polls ppoll Roles prole
Welcome Messages pwelcome Server Updates pserver
Goodbye Messages pgoobye Member Updates pmember
Tickets ptickets Bot Logs ppolaris

Once you think your channels have been setup correctly, you can try one of the channel dependant command like /suggest or /warn , if the bot responds in the correct channel, you can proceed to the next step, if not, please contact Support!
