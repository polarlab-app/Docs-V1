Systems ➜ Moderation

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The moderation system is equipped with every moderation command you might need, from warns to bans! It is also mainly focused on, well, moderating members :D Our moderation system is mainly centered around default permissions which a role needs to have. Everything can again be found in the table below. We recommend taking some type to setup the permissions to know you did it correctly. When you think you're done, try running /warn to verify everything is alright!

Command Description Special Permissions
/ban {member} {reason} Bans a member Banning
/unban {member} {reason} Unbans a member Banning
/kick {member} {reason} Kicks a member Kick
/warn {member} {reason} Warns a user Change Nickname
/warns {member} Displays a members warns none
/clearwarns {member} Clears a members warns Manage Nicknames
/mute {member} {reason} Mutes a member Mute Members
/unmute {member} Unmutes a member Mute Membersr
/vmute {member} {reason} Voice mutes a member Mute Members
/vunmute {member} Voice Un-mutes a member Mute Members
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

All commands in this system require some special permissions, but wait! How do you let your members change nicknames when the /warn command requires the permission? You can use our /nick command which is a part of the utility system!
