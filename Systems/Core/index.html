Systems ➜ Core

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The core system is the main system of the bot. If this system is disabled, the bot will stop working. The Core system includes all of the core functions of the bot and core commands like /ping which we used previously! All commands available as the bot core are available in the table below. All commands are equipped with a special permission a user has to have!

Command Description Special Permissions
/ping Gets the bots ping none
/uptime Gets the bots uptime none
/stop Stops the bot Bot Owner
/restart Restarts the bot Bot Owner
/enable {system} Enables a specific bot system Bot Developer
/disable {system} Disables a bots system Bot Developer
/maintenance {status} Updates the bots maintenance status Bot Developer
/bot-setup Sets up the bot Bot Developer
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Most of the commands in this system require special permissions only available to ASC Studios members which means you cannot use them as an ordinary user. These commands however are just administrative and do not limi the functionality of the bot in your server!
